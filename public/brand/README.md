# Logo — كيف تستبدل اللوجو الحقيقي

الموقع يستخدم حالياً لوجو SVG مرسوماً (ميدالية نحاسية بفنجان قهوة). لاستخدام
لوجو لوسيل الرسمي:

1. ضع ملف اللوجو هنا باسم **`logo.svg`** (مفضّل) أو **`logo.png`**.
2. افتح `src/components/ui/Logo.tsx` وغيّر:
   ```ts
   const USE_BRAND_FILE = false;
   ```
   إلى `true`. إن كان الملف PNG غيّر أيضاً `BRAND_FILE = "/brand/logo.png"`.

سيتبدّل اللوجو في كل مكان دفعةً واحدة: الشريط العلوي، التذييل، شاشة الافتتاح،
وأيقونة المتصفّح.

## مواصفات
- شعار **مربّع** (نسبة 1:1) يظهر داخل دائرة بقُطر 40–88px.
- خلفية شفّافة (transparent) مفضّلة.
- إن كان لديك favicon منفصل، استبدل `src/app/icon.svg`.
